# 💡 Why Vlainter?

> “이 질문… 내 이력서를 보고 물어본 걸까?”

많은 면접 준비 서비스는  
단순한 질문 생성에 그칩니다.

Vlainter는 다릅니다.

- 🎯 기술 스택 기반 실전 면접
- 📄 이력서 기반 개인화 질문
- 🤖 Gemini 기반 모범 답안 생성
- 📊 논리·깊이·키워드 분석 피드백
- 🔁 세션 저장 및 복기 기능
